# CLI Tools (work in progress!)
This repository, createad last month (February 2026), lists several CLI tools that should be used from everyone who has a PC. The purpose of these "tools" is to simplify users experience using a personal computer

## Depencies
Literally no one depencies. The only "work table" you need to have is a command line interface (CLI) where you can execute powershell or bash scripts (ps1 or sh format file).

## how to use them?
To use all these scripts, without create a github account and install git command line on your pc, you should navigate to **https://github.com/Philder1523/LinuxTools**. Then, clicking on **Code** button, you will able to download this repository as a **ZIP** file. Before to download these scripts, remember to switch the branch from **main** to **development**
